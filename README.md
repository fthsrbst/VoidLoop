## ğŸ® About the Project

**VoidLoop** is an anomaly-themed escape game set in an office environment. The player, as an office worker caught in a mysterious anomaly, tries to escape in an endless loop. Developed with Unity game engine, this project aims to deliver an atmospheric horror-thriller experience.

### Story
During an ordinary workday, our character encounters strange events in the office. Reality begins to warp, and they find themselves trapped in a repeating anomaly loop. The only goal: Escape and survive.

## ğŸ¯ Gameplay

- **Endless Runner Mechanics**: The player progresses in an endless loop, constantly on the move
- **Anomaly System**: Unexpected changes and mysterious events in the office environment
- **Escape Mechanics**: Overcoming obstacles, avoiding dangers
- **Atmospheric Horror**: Tension-filled office environment and psychological elements

- **Office Environment**: Realistic and atmospheric office design
- **Difficulty Scaling**: Increasing difficulty as the game progresses

- C# based game mechanics
- Custom shader implementations (ShaderLab)

4. Avoid anomalies and try to stay alive
5. Edit game object properties from the Inspector panel
